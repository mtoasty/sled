Update 0.2.0

HUD Revamp:
- Revamped sled spawning
- Revamped settings: not fully functional, removed old settings
- Revamped spectating: revised ui
- Revamped map and added three new locations: Bunny Hill, The Peak, and Ragnarok Access (names subject to change)
- Revamped flashlight: no more levels, server synced
- Revamped keystrokes: no more clashing keybinds, and keystrokes for tilt inputs
- Revamped terminal: refreshed ui and tweaked functionality
- Updated HUD and loading screen for a cleaner, new design

Time trial revamp:
- Completely redesigned time trials
- Time trial rank times have changed, and are likely to change in the future for balance
- New start and finish UI
- New UI-integrated leaderboard
- Physical leaderboards now store the correct, legacy records
- Added new disqualification screen from which you can teleport back to start
- XP awards have been changed, only reflecting rank and trial difficulty

Sled customization:
- Completely redesigned the sled shop UI
- New editable stats for sleds: gyro strength, pitch/roll multipliers, steer angle, steer speed, yaw strength (some are wip)
- Players can now choose the colour of their sled, if applicable, with an advanced colour picker

Datastore rework:
- Completely reworked datastore for improved safety and compatibility
- Levels and XP will be reset as the XP/Level curve has been changed (Legacy level leaderboard remains)
- Time trial times have been reset because of new time trial system and sled physics
- Cosmetic sleds like Currents and Santa's sleighs are preseved along with moons
- Game should no longer get stuck on the loading screen

Time Rework:
- Changed the system for calculating atmosphere conditions
- Added a clock
- Time is now settable by admins (instead of just changing speed)

Fixes:
- Players and their sleds can no longer collide with each other
- Fixed many issues involving resetting
- Fixed issues regarding teleporting: teleporting is now seamless even if seated
- The dual sled no longer is affected by the physics of the passenger

Changes:
- Fixed and balanced sound region volumes
- Reworked random jumpscares
- Some new trails
- Work in progress Mahjong table
- Removed experimental player list
- Sleds now have particle sprays
- Added level up sound
- Many small bugfixes/cleanup

Coming in the future:
- Racing time trials with multiple Players
- Full settings
- Full mobile/controller support
- Tutorial
- More ways to earn XP
- Sledding sounds, music, ambient sounds
- New physical leaderboards for new stats

Report any bugs to #bug-reports, thank you all for waiting for this update!
